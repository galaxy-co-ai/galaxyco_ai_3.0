## Topic Generator

The Topic Generator helps you discover compelling article ideas based on a rough concept.

### How to Use

1. Navigate to Article Studio and select **Topic Generator**
2. Enter your topic area or keyword (e.g., "AI productivity tools")
3. Click **"Generate Ideas"**
4. Review the 3-5 generated topics, each with:
   - **Title** - A compelling, SEO-friendly headline
   - **Description** - What the article will cover
   - **Why It Works** - Analysis of why this topic will resonate
   - **Suggested Layout** - The best article template for this topic
   - **Category** - Content category recommendation

### Actions for Each Topic

- **Save to Topic Bank** - Store for later use
- **Start Writing** - Begin the article creation flow immediately
- **Edit** - Customize the title or description before starting

### Content Gap Analysis

The Topic Generator analyzes your existing blog content to:
- üü¢ **Identify gaps** - Topics you haven't covered yet (shown with "Fills Gap" badge)
- üü° **Flag similarities** - Warn if a topic is too similar to existing posts
- üìä **Show performance data** - Which topic types perform best for your audience

## Brainstorm Mode

Brainstorm Mode is a conversational interface for developing scattered thoughts into structured content.

### When to Use Brainstorm Mode

- You have multiple ideas but aren't sure which angle to pursue
- You want AI to ask clarifying questions before writing
- You're exploring a complex topic and need to narrow focus
- You have raw notes or observations to organize

### How It Works

1. Select **Brainstorm Mode** from Article Studio
2. Share your initial thoughts, notes, or questions
3. AI asks clarifying questions to understand your goals
4. Continue the conversation until you've found your angle
5. Click **"Generate Outline from Conversation"** to create a structured outline

### Best Practices for Brainstorming

- **Start loose** - Don't worry about structure initially
- **Share context** - Tell AI about your audience, goals, or constraints
- **Answer honestly** - AI's questions help narrow the focus
- **Iterate** - You can always adjust the generated outline

## AI-Assisted Writing

The Tiptap-based editor includes powerful AI writing features.

### AI Command Palette

Open the command palette in two ways:
- Type `/` at the start of a new line
- Press `Cmd+J` (Mac) or `Ctrl+J` (Windows) anywhere

**Available Commands:**

| Command | Description | Shortcut |
|---------|-------------|----------|
| **Continue writing...** | AI continues from cursor position | `Cmd+Enter` |
| **Rephrase** | Reword selected text | - |
| **Expand with detail** | Make selection more detailed | - |
| **Make more concise** | Shorten selected text | - |
| **Find source for this** | Search for citations | - |
| **Suggest image here** | Get AI image suggestions | - |

### AI Selection Menu

When you select text, a floating toolbar appears with quick actions:
- **Improve** - Enhance clarity and impact
- **Rephrase** - Alternative wording
- **Shorten** - Reduce word count
- **Source** - Find citations for the claim

### Inline Suggestions

When AI generates content:
1. A suggestion panel appears with the generated text
2. Review the suggestion
3. Press **Tab** or click **Accept** to insert
4. Press **Esc** or click **Dismiss** to reject
5. Click **Regenerate** for a different version

### Continue Writing

1. Position your cursor where you want to continue
2. Press `Cmd+Enter` (Mac) or `Ctrl+Enter` (Windows)
3. AI analyzes context and continues your thought
4. Accept or reject the suggestion

## Source Verification

Article Studio helps ensure all factual claims are properly sourced.

### Using the Source Panel

1. **Select text** containing a factual claim
2. Click **"Find Sources"** in the Source Panel or selection menu
3. AI searches for relevant sources
4. Review suggested sources with:
   - Title and publication name
   - Relevant quote from the source
   - URL to original content
   - Confidence score

### Managing Sources

- **Add Source** - Manually add a source
- **Verify** - Check if source URL is accessible
- **Insert Citation** - Add "According to [Source]" at cursor
- **Delete** - Remove a source

### Verification Statuses

| Status | Meaning |
|--------|---------|
| ‚úÖ **Verified** | Source URL confirmed accessible |
| ‚è≥ **Unverified** | Not yet checked |
| ‚ùå **Failed** | URL inaccessible or content mismatch |

### Pre-Publish Verification

Before publishing, the Pre-Publish Checklist shows:
- Total number of sources
- How many are verified
- Which sources need attention

## Image Generation

Create custom images using DALL-E 3 directly within Article Studio.

### Featured Image

1. In the editor sidebar, find **Featured Image**
2. Click **"AI Generate"** to open the image modal
3. Or **Upload** from your computer
4. Or **Paste URL** for an existing image

### AI Image Modal

**Settings:**

- **Description** - What you want in the image
- **Style** - Professional, Illustrative, Minimalist, Abstract, or Photorealistic
- **Size** - Square (1024√ó1024), Landscape (1792√ó1024), or Portrait (1024√ó1792)

**Actions:**

1. Enter a description or click **"Suggest from content"** for AI-generated prompts
2. Click **"Generate"** to create images
3. Select your preferred option
4. Click **"Insert Selected"** to use the image

### In-Article Images

1. Position cursor where you want an image
2. Open command palette (`/` or `Cmd+J`)
3. Select **"Suggest image here"**
4. AI analyzes surrounding content and opens the image modal with context

## Voice Profile

Train AI to match your brand's unique writing style.

### Accessing Voice Profile

Navigate to **Settings** ‚Üí **Voice Profile** (or `/admin/settings/voice-profile`)

### Analyzing Your Style

1. Click **"Analyze My Blog"**
2. AI examines your published posts to extract:
   - **Tone Descriptors** - Words that describe your style (e.g., "friendly", "technical")
   - **Example Phrases** - Patterns you commonly use
   - **Phrases to Avoid** - Expressions that don't fit your voice
   - **Sentence Length** - Your typical sentence structure
   - **Structure Preferences** - How you organize content

### Customizing Voice Profile

Edit any extracted element:
- Add or remove tone descriptors
- Update example phrases
- Modify structure preferences
- Click **"Save Changes"** to persist

### How Voice Profile Works

When generating content, AI includes your voice profile in its prompts, ensuring:
- Consistent tone across articles
- Brand-appropriate language
- Familiar sentence structures
- Avoided patterns you don't use

## Pre-Publish Checklist

Before publishing, review the comprehensive quality checklist.

### Opening the Checklist

Click **"Publish"** in the editor to open the Pre-Publish Checklist.

### Check Categories

**Content Quality:**
- Title score (word count, power words, emotional triggers)
- Introduction hook detection
- Word count sufficiency
- Actionable takeaways present

**SEO:**
- Meta title length (optimal: ‚â§60 characters)
- Meta description length (optimal: 120-160 characters)
- URL slug validity
- Focus keyword presence

**Sources & Citations:**
- Total source count
- Verification status
- Source URL availability

**Visual Elements:**
- Featured image set
- Content images included
- Alt text on all images

### Auto-Generate SEO

Click **"Generate SEO"** to automatically create:
- Meta title from article title
- Meta description from content
- URL slug from title
- Suggested focus keyword

### Preview Modes

- **Mobile Preview** - See how it looks on phones
- **Desktop Preview** - Standard desktop view
- **Launchpad Preview** - Open actual preview in new tab

### Google & Social Previews

The checklist shows:
- **Google SERP Preview** - How your article appears in search results
- **Social Share Preview** - How it looks when shared on social media

### Publishing

1. Review all checks
2. Acknowledge any critical issues (if proceeding anyway)
3. Click **"Publish Article"**

## Troubleshooting

### AI Generation Fails

- Check your internet connection
- Try regenerating with a different prompt
- Refresh the page and try again

### Images Won't Generate

- Ensure your prompt doesn't violate content policies
- Try a simpler description
- Check if image generation quota is available

### Sources Not Found

- Try more specific search terms
- Verify the claim is factually verifiable
- Consider adding sources manually

### Voice Profile Not Applying

- Ensure you've saved your voice profile
- Check that you have published posts for analysis
- Try re-analyzing your blog
