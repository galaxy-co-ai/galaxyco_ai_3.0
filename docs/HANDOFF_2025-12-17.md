## Documentation Philosophy

The new system follows these principles:

1. **Lean over comprehensive** ‚Äî Only what's needed to start work
2. **Truth over aspiration** ‚Äî Document current state, not goals
3. **Actionable over descriptive** ‚Äî What to do, not what was done
4. **Verified over assumed** ‚Äî Mark unknown status as ‚ö†Ô∏è 
5. **Current over historical** ‚Äî Update frequently, archive old
